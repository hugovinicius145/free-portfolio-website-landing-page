## Instalação

Primeiro, execute o servidor de desenvolvimento:

```bash
npm run dev
# or
yarn dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador e veja o resultado.

Obs: É preciso ter o [Node JS](https://nodejs.org/) instalado no seu computador.
